## 项目 1：多模态视频问答（VideoQA）与视频对话大模型（从训练到上线闭环）

### 背景与目标
- 面向「视频内容理解 + 交互式问答」场景，构建支持视频/图像/文本（可扩展音频）的统一多模态大模型能力，实现事实问答、时序推理、细粒度描述与多轮对话。

### 我的职责
- 负责端到端方案：模型架构设计、训练范式（对齐预训练 + 指令微调）、数据链路与评测系统建设、推理提效与上线质量闭环。

### 技术方案（可被深挖的细节）
- **视频表征与输入预算控制**
  - 采用分段采样（uniform/关键帧/shot 分割）获取 \(T\) 帧，并将每帧切 patch 得到 \(T \times N\) 视觉 token；通过 token budget 将输入控制在可训练/可部署范围内。
  - 针对长视频引入“**先检索片段后生成**”策略：视频分段→轻量检索器召回 Top-K 片段→仅将片段 token 送入 LLM 生成，降低注意力成本并提升相关性。
- **视觉-语言对齐（Alignment）**
  - 采用 CLIP/VideoMAE 等视觉编码器抽取视觉特征；通过 **MLP Projector / Q-Former / Cross-Attention Adapter** 映射至 LLM embedding 空间，并进行消融对比（稳定性、表达能力、成本）。
  - 训练范式：先做“对齐预训练”（让视觉 token 可被 LLM 消化），再做“视频指令微调”（让模型按指令正确使用视频信息回答），显著降低“看不懂但硬编”。
- **跨模态推理与时序对齐**
  - 将问题类型分桶（事实/时序/计数/比较/因果），针对时序类引入时序位置编码、分段记忆（segment memory）与 temporal grounding（答案与片段绑定）。
  - 构建 hard negative（相似实体/相似片段但不同事件），提升模型在“必须依赖视频证据”时的鲁棒性。
- **多模态数据链路（挖掘 + 合成 + 质检）**
  - 从业务视频挖掘高信息片段（高运动/语义变化/字幕密集/高 ASR 置信度），并构造指令数据（问答、描述、对话、多轮一致性样本）。
  - 长尾事件通过 cut-paste/仿真/生成式合成补齐，并配套过滤（伪影检测、ASR 置信门控、去重）降低合成偏差。
- **评测体系与迭代闭环**
  - 离线：按能力维度拆分评测集（事实、时序、细粒度、幻觉/拒答、安全），同时记录“召回片段正确率（grounding recall）”与“答案正确率/一致性”。
  - 在线：埋点采集问题类型、召回片段、生成答案、拒答原因；建立回流机制持续补齐评测覆盖与长尾样本。

### 结果与影响（写法示例，替换为真实数据）
- 离线指标：VideoQA Accuracy 提升 **X%**；grounding 指标提升 **Y%**；细粒度描述质量提升 **Z%**（或人工偏好胜率 +Xpp）。
- 成本与体验：长视频平均推理成本降低 **A%**；端到端时延降低至 **<B ms>**；线上误答/幻觉率下降 **C%**。
- 复用价值：沉淀可复用数据链路与评测平台，支撑多版本基础模型快速回归、灰度与上线。

## 项目 2：音视频交互理解（ASR + A/V 融合）与端到端交互体验优化

### 背景与目标
- 面向「语音提问 + 视频内容理解」的交互场景，在噪声、口音、多说话人等条件下提升可用性与稳定性。

### 我的职责
- 负责音频侧建模、A/V 融合策略、对齐与数据治理，以及推理链路的时延与稳定性优化。

### 技术方案（可被深挖的细节）
- **语音到文本桥接 + 时间戳对齐**
  - 使用 Whisper/自研 ASR 输出转写与时间戳；低置信片段使用纠错/回流机制；对字幕/ASR 与视频片段做对齐，构建“问题—证据片段”监督。
- **音视频融合策略（消融与落地）**
  - 评估早期融合（feature concat）、晚期融合（logit merge）、交叉注意力融合（A↔V cross-attn）三类方案；以线上稳定性/成本为约束选择可控方案并保留离线最优做后续迭代。
- **噪声治理与鲁棒性**
  - 对背景音乐、重叠说话、口音误识别等进行混合噪声增强与置信度门控，减少 ASR 错误传播到视频理解链路。

### 结果与影响（写法示例）
- 交互成功率提升 **X%**，噪声/多说话人场景误答下降 **Y%**；端到端时延下降 **Z%**，满足交互体验阈值。

## 项目 3：多模态大模型训练工程与推理加速框架（支撑快速迭代与上线）

### 背景与目标
- 多模态大模型训练成本高、推理吞吐受限，需要完善的训练—评测—部署工程体系，支撑模型持续迭代与线上稳定交付。

### 我的职责
- 负责分布式训练配置与吞吐优化、显存优化、推理加速与量化编译、质量回归与灰度发布。

### 技术方案（可被深挖的细节）
- **分布式训练与显存优化**
  - 使用 FSDP/DeepSpeed ZeRO（按模型规模选择 stage），配合混合精度、梯度累积、gradient checkpointing 控制峰值显存；基于 profiler 优化通信与 overlap。
- **参数高效微调（PEFT）**
  - LoRA/QLoRA 分层注入（对齐层/跨注意力层/LLM 部分层），先冻住视觉编码器稳定表征，再逐步解冻扩展可训练模块，降低对齐崩坏风险。
- **推理加速（视觉 + LLM + 调度）**
  - 视觉侧优先编译加速（ONNX/TensorRT）、特征缓存与批处理；LLM 侧采用 KV cache、token budget、分段推理；调度侧使用动态 batching 与并发队列提升吞吐。
  - 量化（INT8/FP8）配套校准集与精度回归，确保关键能力维度无明显回退。
- **质量回归与上线闭环**
  - 建立固定回归集（能力维度拆分）与灰度看板（拒答/幻觉/时序错误/召回错误），支持快速定位问题与回滚兜底。

### 结果与影响（写法示例）
- 训练吞吐提升 **X 倍**、单次迭代成本降低 **Y%**；推理时延下降 **Z%**、吞吐提升 **W%**；上线质量回归周期从 **N 天** 降至 **M 小时**。
