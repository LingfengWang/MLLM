## 🎯 核心论文解读

### 1. CLIP (2021) - 对比语言 - 图像预训练

**核心贡献**:
- 对比学习预训练图像 - 文本对
- 零样本图像分类能力
- 开启视觉 - 语言预训练新范式

**架构**:
```
图像编码器 (ViT/ResNet) → 图像特征
                              ↓
                        对比损失
                              ↑
文本编码器 (Transformer) → 文本特征
```

**影响**: 为后续所有 MLLM 提供视觉 - 语言对齐基础

---

### 2. Flamingo (2022) - 视觉语言少样本学习

**核心贡献**:
- 在预训练 VLM 上实现少样本学习
- Perceiver Resampler 架构
- 交叉注意力机制融合视觉 - 语言

**架构**:
```
冻结的视觉编码器 (CLIP)
        ↓
Perceiver Resampler (交叉注意力)
        ↓
冻结的语言模型 (LLM)
```

**影响**: 展示了冻结大模型 + 轻量适配的有效性

---

### 3. BLIP-2 (2023) - 自举图像 - 语言预训练

**核心贡献**:
- Q-Former 轻量查询架构
- 两阶段预训练策略
- 高效的视觉 - 语言对齐

**架构**:
```
图像编码器 (ViT)
    ↓
Q-Former (可学习查询)
    ↓
冻结的 LLM (OPT/Vicuna)
```

**影响**: 降低 MLLM 训练成本，提升效率

---

### 4. LLaVA (2023) - 大型视觉 - 语言助手

**核心贡献**:
- 端到端视觉 - 语言指令微调
- 简单的线性投影器
- 开源社区广泛采用

**架构**:
```
CLIP ViT → 线性投影 → LLM (Vicuna)
```

**影响**: 推动开源 MLLM 发展，成为事实标准

---

### 5. Qwen-VL (2023) - 多功能视觉语言模型

**核心贡献**:
- 阿里通义千问视觉版本
- 支持 OCR、检测、定位等多任务
- 中文场景优化

**架构**:
```
ViT → 位置感知投影 → Qwen LLM
```

**影响**: 中文 MLLM 重要代表

---

## 📈 技术演进路线

```
2021: CLIP (对比预训练)
  ↓
2022: Flamingo (少样本学习 + Perceiver Resampler)
  ↓
2023: BLIP-2 (Q-Former + 两阶段训练)
  ↓
2023: LLaVA (端到端指令微调 + 线性投影)
  ↓
2023: Qwen-VL (多功能 + 中文优化)
  ↓
2024+: 更大规模 + 更高效 + 更多模态
```
