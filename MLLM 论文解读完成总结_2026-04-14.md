## 🔑 关键技术对比

### 视觉 - 语言融合方式

| 模型 | 融合机制 | 可训练参数 | 优势 |
|------|----------|------------|------|
| **CLIP** | 对比学习 | 全部 | 简单、零样本 |
| **Flamingo** | Perceiver Resampler | 全部 | 少样本学习 |
| **BLIP-2** | Q-Former | 187M | 高效 |
| **LLaVA** | 线性投影 | 全部 | 简单 |
| **InstructBLIP** | 指令感知 Q-Former | 187M | 指令跟随 |
| **Qwen-VL** | 位置感知适配器 | 部分 | 细粒度 |

### 训练策略

| 模型 | 预训练 | 指令微调 | 冻结策略 |
|------|--------|----------|----------|
| **CLIP** | 对比学习 | 无 | 无 |
| **Flamingo** | 端到端 | 少样本 | 无 |
| **BLIP-2** | 两阶段 | 支持 | 图像 +LLM |
| **LLaVA** | 预训练对齐 | 指令微调 | 仅 ViT |
| **InstructBLIP** | 两阶段 | 指令微调 | 图像 +LLM |
| **Qwen-VL** | 三阶段 | 指令微调 | 部分 |

## 💡 核心洞察

### 设计原则
1. **简单即有效** - LLaVA 的线性投影胜过复杂架构
2. **冻结大模型** - 保留预训练能力，避免灾难性遗忘
3. **指令调优** - 提升泛化和指令跟随能力
4. **高质量数据** - 数据质量比数量更重要

### 技术趋势
1. **从对比到生成** - CLIP→Flamingo→BLIP-2→LLaVA
2. **从全量到冻结** - 训练效率大幅提升
3. **从单任务到通用** - 指令调优实现通用能力
4. **从粗粒度到细粒度** - 定位、OCR 等能力增强

### 未来方向
1. **多模态融合** - 视觉 + 语言 + 音频 +3D
2. **具身智能** - VLA 模型用于机器人控制
3. **世界模型** - 理解物理规律和因果关系
4. **高效推理** - 量化、蒸馏、稀疏化
